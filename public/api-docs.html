🌐 IPQuery

📖 API 接口文档

纯真IP库查询服务 · 全部接口支持 JSON 和 TXT 两种格式
1
综合查询
支持 IP 地址和域名查询，自动识别类型，返回详细地理位置信息。推荐使用此接口进行所有查询。
GET /api/query?q={ip或域名} JSON
GET /api/query.txt?q={ip或域名} TXT
📎 请求示例
/api/query?q=114.114.114.114
/api/query?q=blog.zeruns.com
/api/query.txt?q=114.114.114.114
2
获取本机公网 IP
返回当前请求来源的公网 IP 地址，无需任何参数。
GET /api/myip JSON
GET /api/myip.txt TXT
📎 请求示例
/api/myip
/api/myip.txt
3
输入 IP 查地理位置
根据指定的 IP 地址查询其地理位置信息。仅支持 IP，不支持域名。
GET /api/location?q={ip} JSON
GET /api/location.txt?q={ip} TXT
📎 请求示例
/api/location?q=8.8.8.8
/api/location.txt?q=8.8.8.8
4
获取访问者地理位置
返回服务端看到的请求来源 IP 及其地理位置。通过 Tailscale/代理访问时，显示的是隧道入口 IP（如 100.x.x.x），非客户端真实公网 IP。
GET /api/mylocation JSON
GET /api/mylocation.txt TXT
📎 请求示例
/api/mylocation
/api/mylocation.txt
5
域名解析 IPv4
解析指定域名的所有 IPv4 地址（A 记录），JSON 格式返回数量和 IP 列表。
GET /api/resolve4?q={域名} JSON
GET /api/resolve4.txt?q={域名} TXT
📎 请求示例
/api/resolve4?q=blog.zeruns.com
/api/resolve4.txt?q=blog.zeruns.com
6
域名解析 IPv6
解析指定域名的所有 IPv6 地址（AAAA 记录），JSON 格式返回数量和 IP 列表。
GET /api/resolve6?q={域名} JSON
GET /api/resolve6.txt?q={域名} TXT
📎 请求示例
/api/resolve6?q=blog.zeruns.com
/api/resolve6.txt?q=blog.zeruns.com
7
获取 IP 库信息
返回当前使用的纯真 IP 库的版本号和描述信息。
GET /api/info JSON
📎 请求示例
/api/info
8
IP 库状态
返回当前 IP 数据库的运行状态、版本和文件信息。
GET /api/status JSON
📎 请求示例
/api/status
